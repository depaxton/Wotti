# הוראות הפעלה - Wotti

## כיצד להפעיל את האפליקציה

### שלב 1: התקנת תלויות (פעם אחת בלבד)
```bash
npm install
```

### שלב 2: הפעלת השרת
```bash
npm start
```

### שלב 3: פתיחת הדפדפן
פתח את הדפדפן שלך וגש לכתובת:
```
http://localhost:5000
```

## זהו! 🎉

השרת שלך עכשיו משרת גם את הפרונטאנד וגם את הבקאנד על פורט 5000.

**אין צורך ב-Live Server יותר!**

## הסבר טכני

השרת Express שלך עכשיו:
- ✅ משרת את כל קבצי ה-HTML, CSS, JS, ותמונות

- ✅ מגיש את ה-API routes ב-`/api/*`

- ✅ עובד על פורט יחיד: 5000

כל הקבצים זמינים דרך: `http://localhost:5000/`
